1. Cấu trúc rẽ nhánh

Cấu trúc rẽ nhánh được thực hiện thông qua lệnh if - lệnh này kiểm tra điều kiện logic nếu đúng thì thì hiện một khối lệnh này, nếu sai thì thực hiện khối lệnh khác

1.1 Rẽ nhánh không đầy đủ

Đây là cấu trúc dạng đơn giản, cú pháp như sau

if ($điều_kiện) {
    // Các lệnh khi điều kiện là đúng
}

$điều_kiện là gí trị true hoặc false, nó thường là kết quả của phép toán logic (>, >=, == , <, <=, &&, ||, !)

//nhập vào $a, $b    
if ($a > $b)
    echo "a lớn hơn b";

Ví dụ, lấy ngày tuần (thứ hiện tại), nếu là thứ 6 in ra câu - chúc cuối tuần vui vẻ

$ngaytuan = date('w');
if ($ngaytuan == 5)
{
    echo "Hôm nay là thứ 6 - chúc cuối tuần vui vẻ";
}

1.2 Rẽ nhánh đầy đủ

Cú pháp

if ($điều_kiện) {
    // Các lệnh khi điều kiện là đúng
} 
else
{
    // Các lệnh  thực hiện  khi điều  kiện sai
}

Ví dụ:

// nhập  biến $a    
if ($a > 0) {
    echo  "$a là số dương";
} 
else
{
    echo  "$a là số âm";
}

Làm ví dụ tổng hợp giải phương trình bậc 2 (nhập tham số a, b, c): a x2 + b x + c = 0

2. Cấu trúc lựa chọn

Đây là cấu trúc rẽ nhiều nhánh thực hiện với lệnh switch, cú pháp như sau:

switch (biểu_thức){
    case giá_trị_1:
        Code thực hiện nếu biểu_thức = giá_trị_1;
    break;

    case giá_trị_n:
        Code thực hiện nếu biểu_thức = giá_trị_n;
    break;

    default:
        Code thực hiện nếu tất cả các trường hợp phía trên không xảy ra.
}

Ví dụ:

// nhập biến $tuoi
echo "Chọn 1 - để tính sin", PHP_EOL;
echo "Chọn 2 - để tính cos", PHP_EOL;
echo "Chọn 3 - tính căn bậc 2", PHP_EOL;

$luachon = readline("Chọn: ");

switch($luachon) 
{
    case '1':
        $so = readline('Góc (rad) cần tính sin:');
        $sin = sin($so);
        echo "Sin của góc $so là $sin", PHP_EOL;
        break;

    case '2':
        $so = readline('Góc (rad) cần tính cos:');
        $cos = cos($so);
        echo "Cos của góc $so là $cos", PHP_EOL;
        break;
        
    case '3':
        $so = readline('Số cần tính căn bậc 2:');
        $can = sqrt($so);
        echo "Căn bậc 2 của $so là $can", PHP_EOL;
        break;
    default:
        echo "Không chọn đúng";
    break;    
}

3. Cấu trúc lặp

Các vòng lặp được sử dụng nhằm mục đích thi khối lệnh nào đó nhiều lần. Trong Php cung cấp một số loại vòng lặp:

  • Vòng lặp for lặp lại một khối lệnh một số lần.
  • Vòng lặp while lặp lại khối lệnh nếu điều kiện kiểm tra là đúng.
  • Vòng lặp do ... while đầu tiên thi hành khối lệnh, đến cuối khối lệnh kiểm tra điều kiện lặp lại khối lệnh.

3.1 Vòng lặp for

Thường bạn sử dụng vòng lặp này khi đã biết rõ số lần cần lặp lại:

for (khởi_tạo; điều_kiện; tăng_giảm)
{
    // Code thi hành;
}

Như vậy vòng lặp for có ba tham số điều khiển lặp cách nhau bởi dấu ; là khởi tạo, điều_kiện, tăng_giảm. Sự làm việc như sau:

  1. Khởi đầu thi hành lệnh tại khởi tạo
  2. Kiểm tra biểu thức điều_kiện, nếu là true thi hành khối lệnh rồi sang bước 3, nếu false kết thúc lặp.
  3. Kết thúc khối lệnh, thi hành biểu thức tăng_giảm, lặp lại bước 2

Lưu ý việc viết ba tham số của for, không cận thận có thể dẫn đền vòng lặp vô tận.

Ví dụ lặp lại khối lệnh 10 lần:

for ($i = 1; $i<=10; $i++) 
{
    echo "Vòng lặp thứ $i";
}

Bài tập: Tính tổng các số nguyên dương chẵn nhỏ hơn 20

3.2 Vòng lặp while, do .. while

3.2.1 Vòng lặp while

Với vòng lặp while thì bạn không cần biết số lần khối lệnh thi hành, mà khối lệnh sẽ thi hành khi mà điều_kiện còn là true , cú pháp lệnh for như sau:

while (điều_kiện là true) {
    khối lệnh thi hành;
}

Ví dụ dưới đây, đầu tiên gán $i là 1 ($i = 1). Sau đó lặp lại khối lệnh khi mà $i còn nhỏ hơn 5 ($i<5). $i được tăng thêm một ($i++) mỗi lần khối lệnh thi hành: 

$i = 1;
while ($i < 5) {
    echo "Giá trị là $i", PHP_EOL;
    $i++;
}

Bài tập: dòng vòng lặp while tính tổng các số nguyên dương lẻ nhỏ hơn 10

3.2.2 Vòng lặp do .. while

Với vòng lặp do ... while, khối lệnh sẽ thi hành ngày, sau đó kiểm tra điều kiện, nếu điều kiện đúng thì lại lặp lại khối lệnh. Cú pháp do ... while như sau: 

do 
{
    khối lệnh thi hành;
} 
while (điều_kiện);

Ví dụ: In ra màn hình giá trị của $i, sau đó tăng $i thêm một. Tiếp theo kiểm tra điều kiện, nếu điều kiện đúng ( $i nhỏ hơn hoặc bằng 5) thì lặp lại khối lệnh.

$i = 2;
do 
{
    echo "Số là " . $i, PHP_EOL;
    $i++;
} while($i <= 5);

4. Các lệnh đặc biệt

Câu lệnh continue; và break; dùng để điều hướng vòng lặp.

4.1. Lệnh break

break; được sử dụng trong các vòng lặp for, while, do while, foreach để ngắt ngay lập tức hoàn toàn vòng lặp (thoát ra khỏi vòng lặp). Nếu sử dụng break ngoài vòng lặp có thể dẫn đến kết thúc chương trình.

Ví dụ: Vị dụ vòng lặp sau kết thúc sau khi số 5 được in ra.

for ($i=0;$i<=50;$i++)  
{
    echo $i;
    if ($i== 5) {
        break;
    }
}

4.2. Lệnh continue

Sử dụng từ khóa continue trong vòng lặp để bỏ qua vòng hiện tài và bắt đầu một vòng mới.

Ví dụ sau chỉ in ra các số lẻ (vòng lặp gặp số chẵn sẽ bỏ qua).

for ($i=0; $i<10; $i++) {
    if ($i%2 == 0) {
        continue;
    }
    echo $i . ' '; 
}
