Tham chiếu, tham trị (7)

1. Khái niệm

Tham trị là các đối số trong các hàm, khi truyền giá trị bởi các biến thì nó sẽ tạo ra biến cục bộ trong hàm - biến này lưu giá trị được truyền đến, bản thân biến ngoài hàm không bị ảnh hưởng. Cách khai báo tham số hàm đã biết là tham trị. 

$a = 10;
$b = 20;

function Ham($x, $y) {
    echo "x = $x, y = $y", PHP_EOL;
    $x = 1;   // gán một giá trị khác
    $y = 2;
};
Ham($a, $b);

echo "a = $a, b = $b", PHP_EOL;  // a = 10, b = 20

Tham chiếu thì khi truyền tham số - nó không tạo ra biến cụ bộ mà nó tự động trỏ đến địa chỉ biến truyền vào, dẫn đến thay đổi giá trị biến trong hàm thì kéo theo thay đổi toàn cục

2. Sử dụng tham chiếu

Khai báo biến kiểu tham chiếu cho thêm ký hiệu & trước tên biến

    $a = 10;
    $b = 20;
    
    function Ham(&$x, &$y) {
        echo "x = $x, y = $y", PHP_EOL;
        $x = 1;   // gán một giá trị khác
        $y = 2;
    };
    Ham($a, $b);
    
    echo "a = $a, b = $b", PHP_EOL;  // a = 1, b = 2
